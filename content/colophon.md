##### the site

fomties.com is based off [quartz](https://quartz.jzhao.xyz/), a static site generator that publishes [obsidian](https://obsidian.md/) notes to the web and hosted on [Netlify](https://netlify.com/). heavily based off [aaron's website](https://aarnphm.xyz/).

all content is written in markdown, stored in an obsidian vault, and deployed via git push.
##### the content

everything on this site is written by [me](https://www.ryeones.com), unless otherwise attributed. conversations in [[stories|stories]] maybe slightly edited for clarity.

all opinions are my own and subject to change as i figure things out.
